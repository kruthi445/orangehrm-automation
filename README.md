# Orange HRM Automation Testing

This repository contains automated tests for the Orange HRM web application using Selenium WebDriver and Python.

## Overview

The automation script tests the following workflow:
1. Login to Orange HRM demo site
2. Navigate to the PIM module
3. Add multiple employees
4. Verify employees in the Employee List
5. Logout from the application

## Prerequisites

- Python or Python 3.7 or higher
- Chrome web browser
